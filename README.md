# Kertausprojekti

#### Projektissa käytetyt kirjastot
    - Node.js
    - Express
    - MariaDB
    - EJS
    - nodemon

### Asennus ohjeet

### `npm install`

### Käytettävät skriptit
### `npm start`
### `npm run dev` (nodemon)

Palvelimen osoite
[http://localhost:3000](http://localhost:3000)

Tämä on festivaali sivu, jossa pystyy etsimään eri festivaaleja ja nähdä festivaalien esiintyjät ja kaupungin.
